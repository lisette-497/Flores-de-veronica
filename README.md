# Flores de Verónica

**Flores de Verónica** es una página web artesanal para mostrar y vender flores hechas a mano por Verónica, una artista apasionada por las manualidades con materiales como limpiapipas.

## Características

- Diseño moderno y limpio, adaptable a móviles y pantallas de escritorio.
- Catálogo con imágenes incrustadas en base64.
- Sección de contacto con formulario.
- Botón de WhatsApp directo para pedidos.

## Cómo usar

1. Descarga el archivo `flores_de_veronica_actualizado.html`.
2. Ábrelo en cualquier navegador para ver el sitio localmente.

### Opcional: Publicar en línea

#### GitHub Pages
1. Sube el archivo HTML a un repositorio público en GitHub.
2. Ve a **Settings > Pages** y elige la rama y carpeta raíz.
3. Tu sitio estará disponible en `https://tuusuario.github.io/nombre-del-repo`.

#### Netlify
1. Ve a [https://netlify.com](https://netlify.com).
2. Crea una cuenta y arrastra el archivo HTML al panel de control.
3. Netlify te proporcionará un enlace público automático.
